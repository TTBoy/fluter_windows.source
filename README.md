# QA ImageProcess - 图像采集质检管理系统

## 项目概述

**QA ImageProcess** 是一个基于Flutter开发的专业图像质检管理系统，专为图像数据的采集、处理、质检和全流程管理而设计。系统采用前后端分离架构，Flutter作为跨平台客户端，主要支持Windows桌面应用环境，提供高效、直观的图像质检解决方案。

整个系统分为三个部分：前端，服务端，浏览器图片采集插件。

该文档主要介绍前端部分。

### 主要特性

- 🔐 **多角色权限管理** - 支持管理员、普通用户和质检人员三种角色
- 📋 **完整工作流程** - 从任务创建、分配、执行到质检交付的全生命周期管理
- 🖼️ **图像处理能力** - 支持图像预览、缩放、AI问题生成与答案匹配、图像查重等功能
- 🤖 **AI集成** - 集成大语言模型（Gemini 2.5 Flash）实现智能图像分析
- 📊 **数据分析** - 提供任务进度跟踪和数据统计功能
- 💾 **文件管理** - 支持图像导入导出、批量处理和下载功能

## 功能模块

### 1. 用户认证与权限管理

#### 登录注册系统

- 邮箱密码登录验证（未使用邮箱验证，只要是邮箱格式即可）
- 用户注册和账户激活
- Token-based身份认证
- 自动登录状态保持（SharedPreferences保存用户数据)

#### 角色权限控制

- **管理员 (role=1)**：拥有所有系统权限
  - 用户管理
  - 任务创建和分配以及质检
  - 系统设置
  - 数据统计查看
- **普通用户 (role=0)**：基础操作权限
  - 查看分配的任务
  - 执行质检工作
  - 个人信息管理
- **质检人员(role=2)：**质检和采集权限
  - 查看所有任务
  - 拉取质检任务

### 2. 工作任务管理

#### 任务生命周期

系统支持8种任务状态的完整流转：

```
未采集 → 正在采集 → 采集完成 → 等待质检 → 正在质检 → 质检打回/质检通过 → 等待交付 → 交付完成
```

实际中“等待交付”和“交付完成”两个状态并未涉及，因为并不是通过系统直接交付，甲方有自己独立的系统，这里只需要一个交付导出功能按照交付格式导出。

甲方会反馈回一些不符合要求的图片，一般是一个Excel表格，因此服务端我添加了一个根据md5查询图片的接口，前端可以使用md5值分页批量查询对应的图片信息。

先使用“读取fileName.py"脚本将md5全部导出到一个json文件了，然后在前端的总览界面使用”json批量查询“功能加载json文件查询图片信息。

后续可以实现直接读取甲方反馈的Excel文件进行查询。

#### 任务管理功能

- **任务创建**：管理员可创建新的图像处理任务
- **任务分配**：将任务分配给指定的采集工作人员
- **进度跟踪**：实时查看任务执行进度和状态
- **任务统计**：提供详细的任务数据统计和报表

### 3. 图像质检系统

#### 质检工作台

- **任务队列管理**：显示待质检、进行中、已完成的任务
- **图像查看器**：支持图像缩放、修改标注信息等操作
- **质检表单**：结构化的质检问题和答案记录
- **质检结果**：支持通过、打回、备注等操作

#### 质检流程

- **质检人员拉取质检任务**：质检人员或者管理员可以在任务管理界面拉取已经采集完成并提交后的任务。
- **结果反馈**：质检结果实时反馈给采集人员。

### 4. 系统管理

#### 用户管理（管理员功能）

- **用户列表**：查看和管理所有系统用户
- **角色分配**：修改用户角色和权限
- **批量操作**：支持批量用户操作
- **用户状态管理**：激活、禁用用户账户

#### 系统设置

- **服务器配置**：后端API服务器地址设置
- **AI服务集成**：配置外部AI服务API、密钥和模型选择
- **文件存储**：配置文件上传和存储路径

### 5. 图像处理与分析功能

- **图像预览**：支持图像查看和缩放操作
- **图像查重**：通过SIFT算法识别重复或相似图像
- **AI分析**：利用视觉模型自动生成图片的问题和答案
- **分类管理**：支持图像分类和标签管理

### 特殊功能使用

#### 图像查重功能

查重界面只是对数据库中的所有图片进行了一个简单的对比（比较其MD5值），其功能有限，只能查询出完全相同的图片。

进入查询相似后，可以使用python编写的大模型脚本分析相似图片（纯用AI写的，具体的原理不太清楚）。

**查相似工作流程：**

选择类目后选择暂存的文件夹，点击“插叙”按钮后下载所选分类的图片到文件夹，将文件夹内的所有图片两两互相比较，得到其相似度，将相似度高的分组，然后在界面上显示。

分析速度极慢。

如果后续有相关需求，建议不使用神经网络算法，使用均值方差，结构性相似指数，或者均值哈希，感知哈希等，速度很快。

## 故障排除

### 常见问题

1. **登录失败**
   - 检查网络连接是否正常
   - 验证服务器地址配置是否正确
   - 确认用户凭据正确性
   - 如果使用代理，确保代理设置正确
2. **图像加载失败**
   - 检查文件路径和权限
   - 验证图像格式支持（支持常见格式如JPG、PNG、BMP等）
   - 检查服务器存储是否正常
4. **AI功能不可用**
   - 检查API配置是否正确
   - API经常会出现限流的情况
5. **查重功能失败**
   - 确认查重路径已正确设置
   - 检查相关Python脚本是否可正常运行
   - 验证图像文件是否可访问
